## Convenciones

- **Formato**: PNG (mejor calidad para UI con texto). JPG está OK para fotos de productos.
- **Dimensiones**: ver tabla. Si la captura nativa es mayor (ej Retina), redimensionar a ancho máx 1600px para no inflar el repo.
- **Nombre de archivo**: kebab-case, descriptivo. Sin espacios.
- **Sin datos sensibles**: revisar que no se vean emails, teléfonos reales de clientes, ni el panel del super admin de PocketBase.
